Embedding the Egypt Tourism Chatbot

This page demonstrates how to embed the Egypt Tourism Chatbot in any website.

How to Embed

Add the following code to your website:

<!-- Egypt Chatbot Widget Configuration -->
<script>
  window.egyptChatbotConfig = {
    serverUrl: 'http://localhost:5000',
    language: 'en',
    autoOpen: false
  };
</script>

<!-- Egypt Chatbot Widget Script -->
<script src="http://localhost:5000/widget.js"></script>

The chatbot will appear in the bottom-right corner of your website.

Configuration Options

  • serverUrl: URL of your Egypt Chatbot server
  • language: Initial language (en/ar)
  • autoOpen: Whether to open the chat automatically
  • selector: CSS selector for a container element

You can also place the chatbot in a specific element:

<div id="egypt-chatbot"></div>

And specify the selector in configuration:

selector: '#egypt-chatbot'

About the Chatbot

The Egypt Tourism Chatbot is an AI-powered virtual guide that helps visitors plan their trip to Egypt. It can provide information about attractions, hotels, restaurants, and practical travel tips.

Built with a hybrid architecture combining:

  • React frontend for modern UI
  • Flask backend for robust NLU
  • Multi-language support
  • Context-aware responses

The chatbot can provide information about attractions like the Pyramids of Giza.
